# Ontology Publish

[![Latest Version on Packagist][ico-version]][link-packagist]

A plugin for [Ontology](https://github.com/Jeroen-G/Ontology) to make the documentation look beautiful!

## Installation

Via Composer

``` bash
$ composer require jeroen-g/ontology-publish
```
